1. Etiquetas

Un elemento generalmente tiene una etiqueta de inicio (por ejemplo, < nombre-de-elemento >) y una etiqueta de cierre (por ejemplo, < /nombre-de-elemento >). Los atributos del elemento están contenidos en la etiqueta de inicio y el contenido está ubicado entre las dos etiquetas (por ejemplo, < nombre-de-elemento atributo="valor">Contenido< /nombre-de-elemento >). Algunos elementos, tales como < br >, no tienen contenido ni llevan una etiqueta de cierre. Debajo se listan varios tipos de elementos de marcado usados en HTML

Ejemplo:

  • < html > < / html >
  • < head > < /head >
  • < body > < / body>

2.Atributos.

La mayoría de los atributos de un elemento son pares nombre-valor, separados por un signo de igual «=» y escritos en la etiqueta de comienzo de un elemento, después del nombre de éste. El valor puede estar rodeado por comillas dobles o simples, dejar los valores sin comillas es considerado poco seguro.

Ejemplo:

  • < h1 style= "font - family: Garamound"> < / h1 >
  • < p style= " color : red" > < / p >
  • < body style= "background-color: green">

3.Imagenes

Para agregar imagenes es muy simple, utilizamos el atributo src dentro de la etiqueta de img (< img >) y luego porcedemos a pegar la ubicacion de la imagen en medio de dos comillas.

Ejemplo:

  • < img src = " http://3.bp.blogspot.com/-RRTN9dzd05I/TuU2_IzEylI/AAAAAAAACjY/QH8nvIRF2FE/s200/HAMBURGUESA.jpg ">
